Создание таблиц

Для создания таблиц используйте специальный синтаксис:

||Заголовок 1|Заголовок 2|Заголовок 3|| ||Ячейка 1|Ячейка 2|Ячейка 3|| ||Данные 1|Данные 2|Данные 3||

Результат:

Заголовок 1 Заголовок 2 Заголовок 3
Ячейка 1 Ячейка 2 Ячейка 3
Данные 1 Данные 2 Данные 3

Правила создания таблиц:

  • Каждая строка таблицы начинается и заканчивается символами ||
  • Ячейки разделяются символом |
  • Первая строка становится заголовком таблицы
  • Последующие строки становятся строками данных
  • Количество ячеек в каждой строке должно быть одинаковым

Примеры

Исходный текст:

Привет всем!

Это **важное** сообщение с *разными* стилями.

Код примера:
`print("Hello World")`

Таблица с данными:
||Имя|Возраст|Город||
||Анна|25|Москва||
||Иван|30|Санкт-Петербург||
